# Aji Forum
A proof-of-concept forum allows users to submit, upvote and downvote posts.

---

## Environments
+ Python >= 3.6
+ Django >= 1.11  

## Installation

    pip install -r requirements.txt

## Deployment

    gunicorn ajiforum.wsgi

And you can visit the website at http://127.0.0.1:8000
